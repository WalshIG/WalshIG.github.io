Capstone Project

Capstone is my final project to complete my ICS program at Camosun College. In this project, I worked in a team with two other students and our client sponsors, Proline Property Management. The purpose of this project is to build a web application to track property keys. We were using the MariaDB database, Express API, and React front-end. We also integrated a label printer and barcode scanner to make use of QR codes for tagging and tracking.
